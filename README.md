# Personal Web Profile
A clean, responsive personal portfolio website showcasing my background as a Multimedia Studies graduate and current Computer Science student.

## Features
- **Semantic HTML5:** Built with accessibility and SEO in mind using proper document structure.
- **Responsive Layout:** Optimized for desktop, tablets, and mobile devices using CSS Grid and Media Queries.
- **Interactive Elements:**
  - Dynamic "Fun Fact" generator with CSS pop-in animations.
  - Smooth-scroll "Back to Top" functionality.
  - Sticky navigation for easy site traversal.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Design:** Modern UI with a custom color palette and CSS variables.

## Project Structure
```text
├── images/          # Image assets (profile, hobbies, etc.)
├── index.html       # Main website structure
├── style.css        # Custom styles and animations
└── script.js        # Interactivity and scroll logic
```
